Sobre el Proyecto

Análisis predictivo para optimizar la gestión de cobranzas empresariales

🎯

Objetivo

Este proyecto busca predecir el desvío en días entre la fecha de vencimiento y la fecha real de pago de una factura (delay_days). Utiliza datos históricos de operaciones entre empresas para asistir decisiones financieras.

📊

Dataset

El dataset incluye fechas clave (emisión, vencimiento, pago), montos, tipo de documento, y datos de comprador/emisor.

🔧

Metodología

Se realizó una limpieza y selección rigurosa de variables para asegurar la calidad del modelo predictivo.

Modelado

Procesamiento de Datos

  • Conversión de fechas a formato datetime
  • Extracción de variables como mes y día de semana
  • Eliminación de columnas irrelevantes o de alta cardinalidad
  • One-hot encoding para categorías
  • Imputación de nulos y escalado de variables numéricas
  • Remoción de columnas constantes

Arquitectura del Modelo

Tipo: Red Neuronal Densa
Framework: TensorFlow/Keras
División: 80/20 (Train/Test)
Métrica: MAE (Mean Absolute Error)
Error Final: ~3.4 días
