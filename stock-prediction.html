Stock Prediction

Un proyecto de predicción de valores bursátiles utilizando modelos de Machine Learning.

Sobre el Proyecto

Este proyecto tiene como objetivo predecir los precios de acciones utilizando modelos avanzados de aprendizaje automático como redes neuronales y algoritmos de regresión.

Detalles Técnicos

Para la predicción de valores se han utilizado las siguientes herramientas y tecnologías:

  • Python y bibliotecas como TensorFlow, Scikit-learn y Pandas.
  • Redes neuronales recurrentes (RNN) y modelos ARIMA.
  • Visualización de datos con Matplotlib y Seaborn.
